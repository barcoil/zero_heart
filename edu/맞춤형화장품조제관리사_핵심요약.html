1과목. 화장품법의 이해

★★★ 화장품의 정의 및 유형

화장품 정의

인체를 청결·미화하고 매력을 더하거나, 용모를 밝게 변화시키거나, 피부·모발의 건강을 유지·증진하기 위해 사용하는 물품 (인체에 대한 작용이 경미한 것)

화장품 vs 의약외품 vs 의약품 | 구분 | 작용 | 예시 | |------|------|------| | 화장품 | 경미한 작용 | 스킨, 로션, 립스틱 | | 의약외품 | 경미한 치료/예방 | 치약, 구중청량제, 탈모방지제 | | 의약품 | 치료/진단/예방 | 연고, 알약 |

★★★ 영업의 종류

영업 내용 필요 자격
화장품제조업 화장품 직접 제조 등록
화장품책임판매업 제조·수입 후 유통 총괄 등록 + 책임판매관리자
맞춤형화장품판매업 즉석 혼합·소분 판매 신고 + 조제관리사

등록 = 제조업, 책임판매업 / 신고 = 맞춤형판매업 ← 자주 출제!

★★★ 맞춤형화장품

  • 정의: 고객 개인의 피부 특성에 맞게 즉석에서 혼합·소분한 화장품
  • 혼합 가능: 내용물 + 내용물 / 내용물 + 원료
  • 소분 가능: 내용물을 작은 용기에 나누는 것
  • 금지: 화장품 원료 단독 판매, 책임판매업 등록 없이 제조

조제관리사 의무 - 맞춤형화장품 판매장에 1명 이상 필수 - 혼합·소분 업무만 가능 (완제품 제조 불가) - 개인정보 수집 시 동의 필요

★★★ 화장품 표시·기재사항

필수 기재 (전 성분 포함) - 제품명 / 내용량 / 제조번호 - 사용기한 또는 개봉 후 사용기간 - 가격 / 제조판매업자 상호·주소 - 전 성분 (1% 초과: 많은 순 / 1% 이하: 순서 무관) - 사용 시 주의사항

기재 생략 가능: 50mL 이하 소용량 (일부 항목)

화장품 유형 (13가지 분류)

영·유아용 / 목욕용 / 인체 세정용 / 눈 화장용 / 방향용 / 두발 염색용 / 색조 화장용 / 두발용 / 손발톱용 / 면도용 / 기초화장용 / 체모제거용 / 체취방지용

행정처분 기준

위반 처분

품질검사 미실시 판매 중지

4과목. 맞춤형화장품의 이해

★★★ 피부의 구조

표피 → 진피 → 피하지방

표피 층 (바깥 → 안쪽)

각질층 → 투명층 → 과립층 → 유극층 → 기저층

층 특징
각질층 가장 바깥, 수분 손실 방지, 각질세포+지질
기저층 가장 안쪽, 세포 분열 (멜라노사이트 존재)
진피 콜라겐·엘라스틴 존재, 피부 탄력

피부 유형

유형 특징
건성 수분·유분 부족, 당김, 잔주름
지성 피지 과다, 모공 확대, 여드름
복합성 T존(지성) + U존(건성)
민감성 자극에 예민, 홍조, 가려움
중성 수분·유분 균형, 이상적

피부 분석 방법

  • 문진: 고객 생활 습관, 피부 고민 상담
  • 시진: 육안으로 피부 상태 관찰
  • 촉진: 직접 만져서 피부결 확인
  • 기기 분석: 피부 측정기 (수분, 유분, 탄력 등 측정)

★★★ 혼합·소분 가능 범위

가능한 것 - 책임판매업자로부터 받은 내용물 + 원료 혼합 - 내용물을 소분

불가능한 것 - 화장품 원료만으로 혼합 (완제품 제조 해당) - 사용 금지 원료 사용 - 의약품 성분 추가

위생 관리 - 혼합 전 손 세정 및 소독 - 혼합 도구·용기 위생 관리 - 작업 공간 청결 유지
