  2. 분산이 높으면 과소적합이 발생한다
  3. 편향과 분산은 동시에 낮출 수 있다
  4. 모델 복잡도가 증가하면 편향은 감소하고 분산은 증가한다

✅ 정답: 4번 | 편향↑=과소적합, 분산↑=과적합. 복잡도↑ → 편향↓ 분산↑

  4. 모델의 정확도가 100%일 때만 적용
